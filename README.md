Deutsch-Digest 🇩🇪
A smart tool for summarizing German news articles.

Features
- Summarization: Uses AI (T5-Base) to condense long German texts.
- Translation:Verifies summaries by translating them to English.
- Visuals:Simple and clean user interface built with Streamlit.

Tech Stack
- Python
- Streamlit
- Hugging Face Transformers
- PyTorch
